# Health World - Healthcare Platform

A comprehensive healthcare platform that provides emergency assistance and at-home care services.

## Project Overview

Health World is an HTML-based website that connects patients with healthcare providers, offering:

- **Emergency Assistance**: Quick access to ambulance, fire, and police services
- **At-Home Care**: Booking doctors, nurses, and physiotherapists for home visits
- **Professional Verification**: Credential verification system for healthcare providers
- **User Profiles**: Separate interfaces for patients, healthcare providers, and pharmacies

## Getting Started

1. **Open the Website**:
   - Open `index.html` in a web browser
   - No server setup required - pure HTML/CSS/JavaScript

2. **Navigate the Platform**:
   - **Home**: Access emergency services and at-home care
   - **Profiles**: Register as patient, healthcare provider, or pharmacy
   - **Verification**: Upload documents for professional verification

3. **Test Features**:
   - Try the emergency assistance module
   - Book at-home care services
   - Complete profile registration
   - Upload verification documents
